# 🚌 Trichy Smart Bus Tracker - திருச்சி பஸ் டிராக்கர்

Live bus tracking prototype for Tiruchirappalli (Trichy) city buses.

## 🚀 Live Demo
[View Demo](https://trichy-bus-tracker.vercel.app)

## ✨ Features
- 🗺️ **Live Map** - Real-time bus locations on interactive map
- 🔍 **Smart Search** - Find buses between any two stops
- 🎤 **Voice Search** - Tamil & English voice input
- 👥 **Crowd Status** - Live passenger density
- ⏱️ **ETA Tracking** - Estimated arrival times
- 🚦 **Traffic Layer** - Real-time traffic hotspots
- 🔔 **Live Alerts** - Delays, weather, festival specials
- 🎫 **Ticket Booking** - Quick digital ticket booking
- 📱 **Responsive** - Works on all devices
- 🌐 **Bilingual** - Tamil & English support

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5.3
- Leaflet.js Maps
- OpenStreetMap
